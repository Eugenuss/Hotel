====================================================================================
	Роли
====================================================================================

Админ:
 - Добавлять комнаты;
 - Удалять комнаты;
 - Создавать, изменять, и удалять пользователей.
 
Директор:
 - Статистика по доходам;
 - По кол-ву гостей;
 - По кол-ву свободных мест;
 
Ресепшен:
 - Изменяет статус комнат;(занята\свободна\готовится)
 - Изменяет статус комнаты статус брони комнаты;(забронирована\незабронирована)
 - Создаёт, изменяет и удаляет гостей;

Горничная:
 - Изменяет статус комнаты статус уборки;(прибрана\неприбрана)

	Таблицы
====================================================================================
Комнаты:
- Number // Номер комнаты [302, 1, 2, 405,,,]
- Person //Кол-во мест [1, 2, 3, 4,,,]
- BusyStatus // Статус занятости [занята, свободна, готовится]
- CleanStatus // Статус уборки [прибрана, неприбрана]
- BookingStatus // Статус брони [забронирована, незабронирована]
- Cost // Стоимость [15000, 20000,,,]

Гости:
- FirstName
- Secondname
- MiddleName
- Seria // ограничение 4 цифры
- Nomer // ограничение 6 цифр
- ZaseleneDay 
- ViezdDay
